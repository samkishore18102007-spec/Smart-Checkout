# Smart Checkout

Smart Checkout is a mobile-based shopping system that allows users to scan items and pay directly without waiting in queues.

## 🔗 Live Demo
Click here to view the working prototype:
https://id-preview--c99dd393-1e59-4e6e-b987-4e6bbc3d3691.lovable.app

## Features
- Scan products using mobile
- Automatic cart update
- Digital payment system
- Sensor-based exit security system

## Flow
Login → Scan → Add to Cart → Pay → Sensor Check → Exit

## Security
Sensors at the exit detect unpaid items and generate an alert to prevent cheating.

Includes app screens for cart, scan, and payment.
